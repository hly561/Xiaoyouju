# 腾讯地图API配置说明

## 问题描述
当前项目的定位功能不够准确，主要原因是缺少微信小程序后台的合法域名配置。项目已经集成了腾讯地图SDK并配置了API密钥，但需要完成域名配置才能正常使用。

## 当前配置状态

✅ **已完成的配置：**
- 腾讯地图SDK已集成（`utils/qqmap-wx-jssdk.js`）
- API密钥已配置：`4YNBZ-WOCKG-R2FQP-QTOB3-EFLSS-3LB5F`
- 坐标系已设置为gcj02（国测局坐标系）
- 模块导入路径已修复

❌ **待完成的配置：**
- 微信小程序后台合法域名配置（关键步骤）

## 解决方案

### 1. 微信小程序后台域名配置（必须完成）

**这是当前唯一需要完成的配置步骤！**

1. 登录微信小程序管理后台：https://mp.weixin.qq.com/
2. 进入【开发管理】→【开发设置】
3. 找到【服务器域名】部分，点击【修改】按钮
4. 在【request合法域名】中添加：`https://apis.map.qq.com`
5. 点击【保存并提交】

**重要提示：** 如果不配置此域名，会出现"非业务域名apis.map.qq.com请重新配置"的错误。

### 2. API密钥配置（已完成）

✅ API密钥已在 `config/index.js` 中正确配置：

```javascript
export const config = {
  useMock: true,
  // 腾讯地图API密钥（已配置）
  qqMapKey: '4YNBZ-WOCKG-R2FQP-QTOB3-EFLSS-3LB5F',
};
```

### 3. 启用的功能

配置完成后，以下页面的定位功能将使用腾讯地图的逆地理编码服务：

- **首页定位**：`pages/home/index.js` - 自动获取用户当前城市
- **城市选择器**：`pages/city-selector/index.js` - 精确定位当前位置

### 4. 降级机制

如果未配置API密钥或API调用失败，系统会自动降级到简化的城市判断逻辑，确保基本功能正常运行。

### 5. 注意事项

1. **API配额**：腾讯地图API有调用次数限制，请根据实际使用情况选择合适的套餐
2. **网络权限**：确保小程序已配置网络请求权限
3. **定位权限**：用户需要授权位置信息获取权限
4. **域名配置**：必须在小程序管理后台配置腾讯地图API域名，否则会出现"非业务域名"错误

### 6. 测试验证

配置完成后，可以通过以下方式验证：

1. 打开小程序首页，查看城市定位是否更加准确
2. 进入城市选择页面，点击定位按钮测试定位功能
3. 查看控制台日志，确认腾讯地图API调用成功

## 🔧 定位仍然不准确？故障排除指南

如果完成域名配置后定位仍然不准确，请按以下步骤排查：

### 1. 检查开发工具设置

**开发者工具中的定位本身就不准确，这是正常现象！**

- 开发者工具使用IP定位，精度有限
- 建议使用真机调试或预览功能测试
- 在开发者工具中：设置 → 项目设置 → 勾选"不校验合法域名"

### 2. 真机测试

1. 使用微信开发者工具的"预览"功能
2. 在真实手机上扫码测试
3. 确保手机GPS定位已开启
4. 在户外或窗边测试（室内GPS信号弱）

### 3. 检查API调用状态

打开微信开发者工具控制台，查看是否有以下错误：

```
// 正常调用应该看到：
[腾讯地图API] 逆地理编码成功

// 如果看到错误：
非业务域名apis.map.qq.com请重新配置  ← 域名配置问题
API调用失败  ← API密钥或网络问题
```

### 4. 验证配置清单

- [ ] 微信小程序后台已添加 `https://apis.map.qq.com`
- [ ] API密钥正确：`4YNBZ-WOCKG-R2FQP-QTOB3-EFLSS-3LB5F`
- [ ] 小程序已获得位置权限
- [ ] 使用真机测试而非开发者工具

### 5. 强制刷新配置

1. 微信开发者工具：项目 → 清除缓存 → 清除全部
2. 重新编译项目
3. 重新预览到手机测试

### 6. 检查定位权限

确保在 `app.json` 中已配置位置权限（已配置）：

```json
"permission": {
  "scope.userLocation": {
    "desc": "你的位置信息将用于小程序定位"
  }
}
```

### 7. 联系技术支持

如果以上步骤都无法解决，可能是以下原因：

- API密钥配额不足
- 腾讯地图服务异常
- 特定地区定位精度问题

建议联系腾讯地图技术支持：https://lbs.qq.com/

> ⚠️ **重要提醒：** 开发者工具中的定位精度本身就有限，请务必在真机上测试！
